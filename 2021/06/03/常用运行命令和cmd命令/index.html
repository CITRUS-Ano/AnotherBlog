常用运行命令和cmd命令以及快捷键

运行 : Win键+R
cmd : Win键+R->键入cmd

Quick Start

运行命令

1.cmd命令提示符 : cmd
2.注册表 : regedit
3.服务 : services.msc
4.设备管理器 : devmgmt.msc
5.OBDC数据源管理器 : Odbcad32
6.控制面板 : control

cmd命令

1.目录操作
  • 切换磁盘: d: (进入D盘)
  • 进入文件夹: 文件夹名/子文件夹名…
  • 返回根目录: cd\
  • 回到上级目录: cd ..
  • 新建文件夹: md test
2.进程管理
  • 显示正在运行进程: tasklist
  • 查看某一端口进程: netstat -ano|findstr “端口号”
  • 结束进程(按进程号): taskkill | findstr “进程(pid)号”
  • 杀死所有进程: taskkil /f /t /im “进程id或进程名称”
3.服务管理
  • 显示正在运行服务: net start
  • 启动指定服务: net start 服务名
  • 停止指定服务: net stop 服务名

小tips:

  • win10快捷键

    • 截图: win键+shift+S
    • 粘贴板记录: win键+V

  • 常用cmd命令保存到记事本中,将后缀改为.bat

  • 启动目录位置:

C:\Users\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

  • Win10隔离区位置:

C:\ProgramData\Microsoft\Windows Defender\Quarantine
