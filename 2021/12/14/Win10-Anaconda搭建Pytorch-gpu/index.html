Win10 Anaconda搭建Pytorch-gpu

QUICK START

1、新建Pytorch-gpu环境

​ 在Anaconda Nevigator中CREATE，取名为pytorch-gpu

2、打开Anaconda Prompt

​ 1）进入pytorch-gpu环境

​ 2）查看已安装的module

​ 3）若有pytorch（如cpu）版本需先进行卸载

3、安装CUDA

​ 1）选择合适的CUDA版本（当前pytorch官网支持：CUDA 10.2和CUDA 11.3）

多版本CUDA网址

选择安装CUDA 11.3.1，并将CUDA安装在默认位置中

​ 2）添加CUDA环境变量（其他版本将v11.3改为v.xx即可）

  • 直接在系统变量中新建
变量 值
CUDA_LIB_PATH %CUDA_PATH%\lib\x64
CUDA_PATH C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.3
CUDA_PATH_V11_3 C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.3
CUDA_SDK_BIN_PATH %CUDA_SDK_PATH%\bin\win64
CUDA_SDK_LIB_PATH %CUDA_SDK_PATH%\common\lib\x64
CUDA_SDK_PATH C:\ProgramData\NVIDIA Corporation\CUDA Samples\v11.3
  • 在cmd中测试CUDA是否安装成功
  • 查看环境变量是否搭建成功

​ 3）安装cuDNN（需要和CUDA的版本对应）

cuDNN官网下载（需注册)

  • 下载cuDNN v8.2.1 for CUDA 11.x
  • 下载完成后，解压压缩包，将其中的bin、include、lib文件夹中的文件分别放到CUDA目录下的对应文件夹中

​ 4）离线安装GPU版pytorch

离线安装pytorch地址

  • 下载其中的torch和torchvision（cu113：CUDA v11.3 cp38：python3.8）

cu113/torch-1.10.0%2Bcu113-cp38-cp38-win_amd64.whl

cu113/torchvision-0.11.0%2Bcu113-cp38-cp38-win_amd64.whl

  • 将下载后的文件放到env文件夹中对应环境文件夹中（使用pip安装，将pkg_name.whl换为下载文件的文件名）

​ 5）测试gpu-Pytorch是否可以使用
