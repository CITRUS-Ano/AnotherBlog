Win10 Anaconda搭建Pytorch-gpu

发表于 2021-12-14

QUICK START

1、新建Pytorch-gpu环境

​ 在Anaconda Nevigator中CREATE，取名为pytorch-gpu

2、打开Anaconda Prompt

​ 1）进入pytorch-gpu环境

​ 2）查看已安装的module

​ 3）若有pytorch（如cpu）版本需先进行卸载

3、安装CUDA

​ 1）选择合适的CUDA版本（当前pytorch官网支持：CUDA 10.2和CUDA 11.3）

多版本CUDA网址

选择安装CUDA 11.3.1，并将CUDA安装在默认位置中

​ 2）添加CUDA环境变量（其他版本将v11.3改为v.xx即可）

  • 直接在系统变量中新建
变量 值
CUDA_LIB_PATH %CUDA_PATH%\lib\x64
CUDA_PATH C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.3
CUDA_PATH_V11_3 C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.3
CUDA_SDK_BIN_PATH %CUDA_SDK_PATH%\bin\win64
CUDA_SDK_LIB_PATH %CUDA_SDK_PATH%\common\lib\x64
CUDA_SDK_PATH C:\ProgramData\NVIDIA Corporation\CUDA Samples\v11.3
  • 在cmd中测试CUDA是否安装成功
  • 查看环境变量是否搭建成功

​ 3）安装cuDNN（需要和CUDA的版本对应）

cuDNN官网下载（需注册)

  • 下载cuDNN v8.2.1 for CUDA 11.x
  • 下载完成后，解压压缩包，将其中的bin、include、lib文件夹中的文件分别放到CUDA目录下的对应文件夹中

​ 4）离线安装GPU版pytorch

离线安装pytorch地址

  • 下载其中的torch和torchvision（cu113：CUDA v11.3 cp38：python3.8）

cu113/torch-1.10.0%2Bcu113-cp38-cp38-win_amd64.whl

cu113/torchvision-0.11.0%2Bcu113-cp38-cp38-win_amd64.whl

  • 将下载后的文件放到env文件夹中对应环境文件夹中（使用pip安装，将pkg_name.whl换为下载文件的文件名）

​ 5）测试gpu-Pytorch是否可以使用

极客云V2ray客户端搭建

发表于 2021-06-03

极客云官网->商店(购买服务)->下载和教程(V2ray客户端下载)

QUICK START

V2ray客户端配置

1.解压文件, 通过V2ray.exe打开(在任务栏最小化显示),

2.点开后订阅->订阅设置,备注名任取,地址url(官网->下载与教程->V2ray订阅链接),回到V2ray选择订阅->更新订阅

3.关闭v2ray, 来到最小化任务栏,右键V2ray选择服务器选择一个节点,然后选择http代理, 打开全局代理测试

chrome网页插件配置

1.https://chrome.google.com/webstore/detail/proxy-switchyomega/padekgcemlokbadohgkifijomclgjgif 打开链接安装Proxy SwitchyOmega插件,并在扩展程序->Proxy SwitchyOmega->更多操作->固定,将其固定在浏览器左上角

2.点击Proxy SwitchyOmega图标->选项->情景模式->代理,将代理服务器中的代理协议改为sock5,代理服务器改为localhost,代理端口号改为不冲突的端口号(如2333)

3.回到V2ray选择参数设置->Core:基础设置,将本地监听端口改为网页修改为的端口号,然后确定

4.回到 SwitchyOmega,选择auto switch,添加规则列表规则,情景模式选择代理,规则列表格式选择auto Proxy, 规则列表网址粘贴 https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt (内包含常用外网,若有些外网未自动打开Proxy, 可点击SwitchyOmega选择添加条件即可)

5.回到v2ray任务栏最小化, 右键关闭全局代理, 进入百度和google时将鼠标放到SwitchyOmega图标上查看, 国内网站应为直接连接, 国外网站应显示订阅设置中的备注, 即为自动代理成功

ios端配置

1.打开 https://free.shadowrocket.online/ 随便选取一个用户名密码登录app store美区(不要登陆iCloud), 在商城下载shadowrocket

2.用ios端浏览器打开极客云网址, 在首页最下方点击一键导入Shadowrocket配置,会提示跳转到Shadowrocket, 自动导入配置节点,选择节点后启动即可

小tips:

  • ​ 美区账号注册
  1. 打开苹果官网注册账户
  2. 地区选择United States(可使用中国手机号注册, 名字最好写英文)
  3. 注册成功后登录App Store, 根据提示点击并按照以下内容填写表单

常用运行命令和cmd命令以及快捷键

发表于 2021-06-03

运行 : Win键+R
cmd : Win键+R->键入cmd

Quick Start

运行命令

1.cmd命令提示符 : cmd
2.注册表 : regedit
3.服务 : services.msc
4.设备管理器 : devmgmt.msc
5.OBDC数据源管理器 : Odbcad32
6.控制面板 : control

cmd命令

1.目录操作
  • 切换磁盘: d: (进入D盘)
  • 进入文件夹: 文件夹名/子文件夹名…
  • 返回根目录: cd\
  • 回到上级目录: cd ..
  • 新建文件夹: md test
2.进程管理
  • 显示正在运行进程: tasklist
  • 查看某一端口进程: netstat -ano|findstr “端口号”
  • 结束进程(按进程号): taskkill | findstr “进程(pid)号”
  • 杀死所有进程: taskkil /f /t /im “进程id或进程名称”
3.服务管理
  • 显示正在运行服务: net start
  • 启动指定服务: net start 服务名
  • 停止指定服务: net stop 服务名

小tips:

  • win10快捷键

    • 截图: win键+shift+S
    • 粘贴板记录: win键+V

  • 常用cmd命令保存到记事本中,将后缀改为.bat

  • 启动目录位置:

C:\Users\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

  • Win10隔离区位置:

C:\ProgramData\Microsoft\Windows Defender\Quarantine
